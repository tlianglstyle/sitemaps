## Build Setup

``` bash
# install dependencies
// 安装依赖包
npm install

// 开发
npm run dev

// 编译打包
npm run build

// 启动生产页面
npm start
```
新创建页面在src下添加文件夹并创建```pageinfo.json``` 然后```npm run dev``` 即可
